# Grocery Store Management System (GSMS)

The project allows users to:

* Add and manage grocery products
* Create customer orders
* Calculate total order cost automatically
* Delete products dynamically
* View order history and summaries

## Technologies Used

* **Python (Flask)** — Backend server and API development
* **MySQL** — Database management and data storage
* **HTML** — Structure of web pages
* **CSS** — Styling and responsive UI design
* **JavaScript & jQuery** — Dynamic frontend interactions and API integration

## Features

* Product Management Dashboard
* Add/Delete Products
* Dynamic Unit Dropdown
* Order Creation System
* Automatic Price & Total Calculation
* Order History Tracking
* REST API Integration with Flask
* MySQL Database Connectivity

## Database

The application uses **MySQL** to store:

* Product details
* Units of measurement
* Customer orders
* Order items and pricing data

## Backend

The backend is developed using **Python Flask**, providing APIs for:

* Fetching products
* Adding products
* Deleting products
* Creating orders
* Retrieving order history

## Frontend

The UI is built using:

* HTML
* CSS
* JavaScript
* Bootstrap
* jQuery AJAX

to provide a responsive and user-friendly interface.
